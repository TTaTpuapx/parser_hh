Парсер вакансий с сайта hh.ru, использующий официальное API.

Собирает, обрабатывает и сохраняет данные о вакансиях в форматах JSON и CSV.

Подойдёт для анализа рынка труда, учебных проектов или создания собственного мониторинга вакансий.

**Содержание**

* Возможности
* Технологии
* Установка
* Использование
* Пример выходных данных
* Структура проекта

**Возможности**

* Поиск вакансий по ключевым словам (например, «Python разработчик», «Java», «Data Science»).
* Фильтрация по региону (Москва, Санкт-Петербург, Россия и др.).
* Загрузка детальной информации по каждой вакансии:

  * Название, работодатель, город
  * Зарплата 
  * Требуемый опыт, тип занятости, график
  * Ключевые навыки и описание
  * Сбор нескольких страниц результатов 
  * Сохранение результатов в JSON
  * Вывод простой статистики: топ работодателей, самые востребованные навыки.
  * Задержки между запросами
  * Обработка ошибок 403

**Технологии**

* Python 3.10+
* requests
* pandas
* tqdm
* json
* datetime

**Установка**

1. Клонирование репозитория

git clone https://github.com/TTaTpuapx/parser\_hh.git

cd parser\_hh

2\. Создать и активировать виртуальное окружение

python -m venv venv

venv\\Scripts\\activate

3\. Установить зависимости

pip install -r requirements.txt

**Использование**

Запуск парсера с настройками по умолчанию
